# Notepad++ for macOS

这是一个正在进行中的项目，旨在将流行的 Windows 文本编辑器 Notepad++ 移植到 macOS 平台。

## 项目结构

- `ScintillaFramework` - Scintilla 的 macOS 框架
- `Lexilla` - 词法分析器库
- `NotepadPlusPlus` - 主应用程序代码
- `Resources` - 图标和其他资源文件
